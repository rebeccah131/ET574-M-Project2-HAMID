# Project-2 

Description:

In this project a desktop application will be designed and implemented by using Python in order to visualize dataset from 
the UCI Irvine Machine Learning Repository. To create this, 
NumPy, Pandas, Matplotlib, and wxPython will be utilized. As a resultm the application will have an interactive GUI, load data from a file, and will display a graph or chart in a pop-up window.
